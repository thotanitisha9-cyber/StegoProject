# StegoProject
A comprehensive steganography platform for securely hiding and extracting secret messages across images, audio, video, text, and pixel data using cryptography, machine learning, and Flask-based web technologies.
The following files and folders were not included in this GitHub repository because they are generated files, large datasets, or large media files:

.vscode/
dataset/
NewDataset/
models/
stego_video.avi
stego_video.mp4
These files can be generated or added separately if required.The source code necessary to run this project is included in this repository.
